What is Lazy Launderette

- A screenshot of the player's starting area.

Lazy Launderette is a small and simple game about searching a launderette for loose change in an attempt to gather enough money to pay to use a washing machine.

- A screenshot of the player picking up someone's washing.

- A screenshot of the cosy area in the Launderette.

The game was made in 48 hours for the 2024 Global Game Jam.

- A screenshot of the outside world at night.

- A screenshot the player placing their laundry in a machine.

I would like to return to this project at a later date, but this version is held together with spaghetti code, so chances are if I did return to it 90% of my time would be spent simply cleaning up code.
